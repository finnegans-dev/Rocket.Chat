# Instalación

* meteor npm install

# Ejecución

* meteor run
* Ingresar a http://localhost:3000 para registro de primer administrador y configuración.

# Rutas

* Registro
* http://localhost:3000/loginGO/{token}&email={email} 

* Login
* http://localhost:3000/registrarGO/{token}&email={email}
